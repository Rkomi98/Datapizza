\(\displaystyle \begin{aligned} &a = 1.0 &&\text{Ricostruzione multisensore}\\ &b = 0.05 &&\text{Uniformità batch}\\ &c = 0.02 &&\text{Consistenza teacher–student}\\ &d = 0.001 &&\text{Allineamento semantico (CLIP)} \end{aligned} \)

📊 Cosa significa ogni componente della Loss:

a = 1.0 - Ricostruzione 🔄

Garantisce che l'AI possa ricostruire accuratamente riflettanza, radar backscatter, altezze LiDAR dai 64 bytes compressi

b = 0.05 - Uniformità 🌐

c = 0.02 - Consistenza 🎯

Sistema Teacher–Student per Data Sparsity

👑 Teacher Model

  • Accesso completo a tutte le bande multisensore
  • Obiettivo: performance ottimale, senza input mancanti

🎓 Student Model

  • Input perturbati: nuvole simulate, drop sensori
  • Obiettivo: imparare robustezza con dati mancanti

Strategie di perturbazione

  • Source dropping: rimozione 30 % Landsat/Sentinel-1 (mai Sentinel-2).
  • Temporal dropping: caduta casuale di timesteps nella sequenza.
  • Forecasting: rimozione degli ultimi 6 mesi di dati.
  • Backcasting: rimozione dei primi 6 mesi di dati.

🎯 Perché è cruciale il Teacher-Student:

Problema reale:

  • 60% immagini coperte da nuvole in zone tropicali
  • Sentinel-1 manca in molte aree (swath gaps)
  • Satelliti vanno offline (es. Sentinel-1B nel 2021)
  • Landsat ha revisit di 16 giorni

Soluzione AlphaEarth:

  • Student addestrato con 30-50% dati mancanti
  • Consistency loss L_cons = (1 - μ·μs)/2
  • Funziona anche con SOLO Sentinel-2
  • Embeddings stabili nonostante input variabili

Risultato: AlphaEarth mantiene R² > 0.5 per evapotraspirazione anche con 70% dati mancanti, mentre altri metodi falliscono completamente (R² < 0).

Sistema Teacher–Student per Data Sparsity

Consistency Loss

\[ L_{\text{cons}} = \frac{1 - \langle \mu,\,\mu_s\rangle}{2} \]

  • Ricostruzione: garantisce che l’encoder non perda riflettanza, back-scatter e altezze LiDAR.
  • Uniformità batch: spinge gli embedding a coprire uniformemente lo spazio (utile per k-NN e clustering).
  • Consistenza T–S: il teacher guida lo student anche con dati mancanti o nuvolosi.
  • Allineamento testo (CLIP): trasferisce conoscenza semantica da testi geolocalizzati (“foresta ombrofila”).
